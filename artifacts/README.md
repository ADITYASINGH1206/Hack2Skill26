# Pre-computed ML Artifacts

This folder holds the pre-computed artifacts generated by the offline indexer.
They are NOT checked into git because they are large binary files.

## How to Generate

**Option A: Google Colab (Recommended, ~5 min on T4 GPU)**
1. Upload `notebooks/role2_pipeline.ipynb` to Google Colab
2. Set runtime to T4 GPU
3. Upload `candidates.jsonl` when prompted
4. Run all cells
5. Download the 4 artifact files and place them here

**Option B: Local CPU (Slow, ~45-60 min)**
```bash
python -m src.ml.indexer candidates.jsonl
```

## Expected Files

After generation, this folder should contain:

| File | Description | Approx Size (100K candidates) |
|------|-------------|-------------------------------|
| `dense_index.faiss` | FAISS inner-product index of BGE embeddings | ~293 MB |
| `bm25_index.pkl` | Pickled BM25Okapi sparse index | ~200 MB |
| `candidates_meta.pkl` | Lean candidate metadata for scoring | ~300 MB |
| `jd_vector.npy` | Pre-encoded JD query vector (768-dim) | ~3 KB |
